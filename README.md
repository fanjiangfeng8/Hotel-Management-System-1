# Hotel-Management-System
Ajax酒店管理系统，包含数据库文件


首页访问地址 ：http://localhost:8080/JavaPrj_9/

首页配置 页面 修改 打开web.xml

修改 即可  默认为userLogin.jsp

<welcome-file-list>
           <welcome-file>/WEB-INF/pages/userLogin.jsp</welcome-file>
</welcome-file-list>

数据库配置 为hotel-hibernate.xml 文件

以上配置 项目可以运行！！！
